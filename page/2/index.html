ML(5)_分类算法_2

朴素贝叶斯算法

应用场景: 文本分类

“朴素”: 特征与特征之间相互独立

贝叶斯公式: P(C|W) = [P(W|C) * P(C)] / P(W)

拉普拉斯平滑系数

  • P(F1|C) = (Ni+α) / N+α*m
    • α为指定系数,一般为1
    • m为统计特征词个数
  • sklearn.naive_bayes.MultinomialNB(alpha=1.0)
    • 朴素贝叶斯分类
    • alpha: 拉普拉斯平滑系数

决策树

非常”朴素”,可理解为if-else

  • 香农定义信息: 消除随机不定性的数据
    • 信息熵: H(X) = - Σ P(xi)logb(P(xi)), b∈R
    • 信息增益: 特征A对训练集D的信息增益g(D,A),定义为集合D的信息熵H(D)与特征A给定条件下D的信息条件熵H(D|A)之差, g(D,A) = H(D) - H(D|A)
  • sklearn.tree.DecisionTreeClassifier(criterion="gini",max_depth=None,random_state=None)
    • 决策树分类器
    • 默认criterion=”gini”(基尼系数),也可选信息增益的熵”entropy”
    • max_depth树的深度大小
    • random_state随机数种子
  • sklearn.tree.export_graphviz(estimator,out_file="tree.dot",feature_names=[","])
    • 决策树可视化,导出DOT格式

随机森林

集成学习方法: 通过建立几个模型组合的来解决单一预测问题,原理是生成多个分类器/模型,各自独立地学习和作出预测,这些预测最后结合成组合预测,因此优于任何一个单分类的做出预测

随机森林是一个包含多个决策树的分类器(少数服从多数)

  • N为训练样本数,M为特征数
    • 1次随机选出1个样本,重复N次,可能出现重复样本
      • 训练集随机
        • BootStrap: 随机有放回抽样
    • 随机选出m个特征,m << M,建立决策树
      • 特征随机

“愚笨的树终会消失,聪明的树脱颖而出”

  • sklearn.ensemble.RandomForestClassifier(n_estimators=10,criterion="gini",max_depth=None,bootstrap=True,random_state=None,min_samples_split=2)
    • 随机森林分类器
    • n_estimators: 决策树数目
    • 默认criterion=”gini”
    • max_depth: 树的深度大小
    • max_features=”auto”: 每个决策树最大特征数量
    • bootstrap: 是否放回抽样
    • min_samples_split: 节点划分最少样本数
    • min_samples_leaf: 叶子节点最小样本数

ML(4)_分类算法_1

分类问题,目标值为类别

sklearn转换器和预估器

  • 转换器
    1. 实例化一个transfer
    2. transfer.fit_transform() 对于文档建立分类词频矩阵,不能同时调用
      • fit() 计算
      • transform() 最终转换
  • 预估器
    1. 实例化一个estimator
    2. estimator.fit(x_train, y_train) 计算
      • 调用完毕,模型生成
    3. 模型评估
      1. 直接比对真实值和预测值
        • y_predict = estimator.predict()
      2. 计算准确率
        • accuracy = estimator.score(x_test,y_test)

K-近邻算法(KNN)

根据”邻居”判断类别

若一个样本在特征空间中的k个最相似(即特征空间中最邻近,欧氏距离等)的样本中的大多数属于某一个类别,则该样本也属于这个类别

k值过小,容易受异常值影响;k值过大,容易受样本不均衡的影响

适用场景: 几千~几万的小数据规模

  • sklearn.neighbors.KNeighborsClassifier(n_neighbors=5,algorithm='auto')
    • n_neighbors=5 : k值
    • KNeighborsClassifier.fit(x_train, y_train):计算,生成模型

模型选择与调优

n折交叉验证: 将拿到的训练数据,分为训练和验证集,数据分n份,其中一份作验证集,经4次测试,每次都更换不同验证集,即得到4组模型的结果,取平均值

超参数搜索-网格搜索(Grid-Search)

有些需要手动指定的参数(如k值),称超参数,手动过程繁杂,需对模型预设几种超参数组合,每组超参数都交叉验证评估,最后选出最优参数组合建立模型

  • sklearn.model_selection.GridSearchCV(estimator,param_grid=None,cv=None)
    • 对预估器的指定参数值进行详尽搜索
    • estimator: 预估器对象
    • param_grid: 预估器参数(dict){“n_neighbors”:[1,3,5]}
    • cv: 指定几折交叉验证
    • fit(): 输入训练数据(GridSearchCV继承了预估器的对象,可以调用方法)
    • score(): 准确率
    • 结果
      • best_params_
      • best_score_
      • best_estimator_
      • cv_results_

ML(3)_特征降维

某些限定条件下,降低随机变量(特征)个数,得到一组”不相关”主变量

特征选择

从原有特征中找出主要特征

  • 方法
    • Filter(过滤式):探究特征本身特点,特征与特征和目标值之间关联
      • 方差选择法: 低方差特征过滤(组内)
      • 相关系数(组间)
    • Embedded(嵌入式):算法选择特征,特征与目标值之间关联
      • 决策树: 信息熵,信息增益
      • 正则化: L1,L2
      • 深度学习: 卷积等

Filter(过滤式)

  • 低方差特征过滤
    • sklearn.feature_selection.VarianceThreshold(threshold=0.0): 删除所有低方差(threshold阈值)特征
      • VarianceThreshold.fit_transform(X): X ndarray数组,返回值: 训练集差异低于threshold的特征将被删除,默认保留所有非0方差特征(相同值特征)
  • 相关系数: 皮尔逊相关系数,反映变量间相关系数密切程度的统计指标
    • r = (nΣxy - ΣxΣy) / [(√nΣx^2 - (Σx)^2) * (√n*Σy^2 - (Σy)^2)]
    • -1 <= r <=1
      • r>0,两变量正相关;r<0负相关
      • |r|=1,两变量完全相关;r=0,无相关关系
      • 0 <= |r| <= 1,两变量存在一定程度相关,|r|越靠近1两变量关系越密切,靠近0相关性越弱
      • |r|<0.4低度相关,0.4 <= |r| < 0.7显著性相关,0.7 <= |r| < 1高度线性相关
    • r = pearsonr(x,y): x,y某两个特征

若特征和特征之间相关性很高

  1. 选一个作为代表
  2. 相同权重,加权求和
  3. 主成分分析

主成分分析

高维数据 -> 低维数据,减少冗余,可能会舍弃原有数据或创建新的变量;
数据维数压缩,尽可能降低原数据的维数(复杂度),损失少量信息

  • sklearn.decomposition.PCA(n_components=None): 数据分解为较低维数空间
    • n_components
      • 小数: 表示保留信息的百分比
      • 整数: 减少到多少特征
    • PCA.fit_transform(X): X ndarray数组, 返回转换后指定维度的array
