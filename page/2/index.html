ML(2)_特征预处理

通过一些函数转换将特征数据转换成更加合适算法模型的特征数据

  • 无量纲化,统一数据规格,避免权重差异过大
    • 归一化: 数据映射到区间(默认[0,1])内,鲁棒性较差,适合精确小数据场景
      • 某一特征: X’ = (X-min)/(max-min), X” = X’*(mX-mi)+mi, ([0,1] -> [mx,mi])
      • sklearn.preprocessing.MinMaxScaler(feature_range=(0,1),...)
        • MinMaxScaler.fit_transform(X): X numpy.ndarray格式数据, 返回转换后形状相同的ndarray
    • 标准化: 数据变换到均值为0,标准差为1范围内,在已有样本足够多的情况下比较稳定,适合现代嘈杂大数据场景
      • 某一特征: X’ = (X-mean)/σ, (列均值mean,列标准差σ)
      • sklearn.preprocessing.StandardScaler()
        • StandardScaler.fit_transform(X): X numpy.ndarray格式数据, 返回转换后形状相同的ndarray

ML(1)_特征提取

任意数据(文本或图像)转换为可用于ML的数字特征

字典特征提取

  • sklearn.feature_extraction.DictVectorizer(sprase=True,...): sprase=True返回sprase矩阵,反之二维数组
    • DictVectorizer.fit_transform(X): X字典或包含字典的迭代器,返回sprase矩阵(one-hot)
    • DictVectorizer.inverse_transform(X): X array数组或sprase矩阵,返回转换前的数据格式
    • DictVectorizer.get_feature_names_out(): 返回类别名称

应用场景

  • 本身数据源就为字典
  • 数据集类别特征比较多
    • 数据集特征 -> 字典
    • DictVectorizer转换

tips: sprase稀疏矩阵只指出非0值的位置,节省内存

文本特征提取

  • sklearn.feature_extraction.text.CountVectorizer(sopt_words=[]): (count计数)返回词频矩阵,sopt_words停用词
    • CountVectorizer.fit_transform(X): X文本或包含文本字符串可迭代对象,返回sprase矩阵
    • CountVectorizer.inverse_transform(X): X array数组或sprase矩阵,返回转换前的数据格式
    • CountVectorizer.get_feature_names_out(): 返回单词列表
    • .toarray(): 查看密集矩阵(二维数组)

中文时使用jieba分词器

一般更倾向于取能体现本篇文章的关键词

TF-IDF文本特征提取,TF-IDF用以评估字词对文章的重要程度: TF-IDF = TF*IDF

  1. TF(词频): 词在本文章出现频率
  2. IDF(逆向文档频率): 词语普遍重要性度量,某词的 IDF = lg(总文件数/含该词的文件数)
  • sklearn.feature_extraction.text.TfidfVectorizer(stop_words=None,...): 返回词的权重矩阵
    • TfidfVectorizer.fit_transform(X): X文本或包含文本字符串可迭代对象,返回sprase矩阵
    • TfidfVectorizer.inverse_transform(X): X array数组或sprase矩阵,返回转换前的数据格式
    • TfidfVectorizer.get_feature_names_out(): 返回单词列表
    • .toarray(): 查看密集矩阵(二维数组)
