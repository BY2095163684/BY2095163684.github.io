Numpy(I)

高效的运算工具,快速处理任意维度数组

  • ndarray属性
    • ndarray.shape 形状(元组)
    • ndarray.ndim 维度(数组嵌套深度)
    • ndarray.size 数量
    • ndarray.itemsize 字节长(1Byte = 8bit,1字节等于8比特位)
    • ndarray.dtype 类型(整型默认int64,浮点型默认float32)

tips: - ndarray.shape -> (a,b,c,…), a为第一层数组元素个数,b为第二层,…,len((a,b,c,…))是维度

二维矩阵: 行与列
三维: 一维内嵌套多个二维

Jupyter Notebook

web版的IPython, 可用于编程/文档/笔记/展示, .ipynb文件格式是用于计算型叙述的JSON文档格式的正式规范

进入jupyter notebook,new一个.ipynb新文件,使用cell操作运行

要关联Anaconda的环境需要

  • 一些快捷键
    • esc 命令模式
      • shift + Enter: 运行当前 IN 行输出 OUT 并跳转下一行 IN
      • ctrl + Enter: 运行当前 IN 并停留
      • Y: code 模式
      • M: markdown 模式
      • A/B: 当前 cell 上/下 添加 IN
    • 编辑模式
      • Tab补全
