Jupyter Notebook

web版的IPython, 可用于编程/文档/笔记/展示, .ipynb文件格式是用于计算型叙述的JSON文档格式的正式规范

进入jupyter notebook,new一个.ipynb新文件,使用cell操作运行

要关联Anaconda的环境需要

  • 一些快捷键
    • esc 命令模式
      • shift + Enter: 运行当前 IN 行输出 OUT 并跳转下一行 IN
      • ctrl + Enter: 运行当前 IN 并停留
      • Y: code 模式
      • M: markdown 模式
      • A/B: 当前 cell 上/下 添加 IN
    • 编辑模式
      • Tab补全
